Conclusions

The Shared version benefits from the MESIF cache coherence protocol, which allows CPU cores to fetch shared data directly from other cores' L1D caches (Forward state) instead of main memory. As matrix size N grows, the Local version suffers from deep-copy overhead and increased cache pressure, making the Shared version progressively faster.
